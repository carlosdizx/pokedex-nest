# Ejecutar en desarrollo

1. Clone the repository
2. Install dependency
```
npm i
```
3. You have Nest CLI installed
```shell
npm i -g @nestjs/cli
```

4. Start the database with docker - docker-compose
```dockerfile
RUN docker-compose.yaml
```

## Stack
* MongoDB
* NestJs
